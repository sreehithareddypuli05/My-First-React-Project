## 🚀 Getting Started

### 1. Clone the repo


git clone https://github.com/sreehithareddypuli05/My-First-React-Project.git
cd My-First-React-Project

2. Install dependencies

npm install

3. Start development server

npm start

4. Build for production

npm run build

5. Deploy to GitHub Pages

npm run deploy
